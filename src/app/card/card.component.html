<vgr-card> används för att skapa kort, ofta i listor som expanderat innehåll. Ett kort har tre delar: Topp, vänster och höger. Dessa fylls med innehåll genom att använda css-klasser. Ett kort innehåller en eller flera <vgr-card-section> som kan expanderas.

Namn Beskrivning Exempel
expanded: boolean Anger om sektionen är expanderad eller ej [expanded]="true"
title: string Sektionens rubrik [title]="'Rubrik'"
subTitle: string Sektionens underrubrik [title]="'Underrubrik'"
readonly: boolean Anger om symbolen för redigeringsläge skall visas eller ej. False = visa. True = dölj. Default är TRUE. [readonly]="true"
