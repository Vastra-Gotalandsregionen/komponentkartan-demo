Templatestruktur

  • <vgr-card>, första elementet som representerar hela kortet
  • <vgr-card-header>, ger titelyta på en vgr-card. Plats för knappar och rubriker för hela kortet.
  • <vgr-card-column>, kolumner i kortet.
    • Om det bara finns en kolumn blir den fullbredd.
    • Om det finns två kolumner så hamnar den första till vänster och den andra till höger.
    • Fler än två kolumner stöds ej.
  • <vgr-card-section>, expanderbar sektion som placeras i en kolumn.

OBS! API nedan avser <vgr-card-section>
Namn Beskrivning Exempel
expanded: boolean Anger om sektionen är expanderad eller ej [expanded]="true"
title: string Sektionens rubrik [title]="'Rubrik'"
subTitle: string Sektionens underrubrik [title]="'Underrubrik'"
readonly: boolean Anger om symbolen för redigeringsläge skall visas eller ej. False = visa. True = dölj. Default är TRUE. [readonly]="true"
