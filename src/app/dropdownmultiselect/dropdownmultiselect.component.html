<vgr-dropdown-multiselect> innehåller en lista där man kan välja flera alternativ.
- Kompletteras med filter och ett "visa alla" alternativ över 20 items.
- Kompletteras med scroll vid över 8 items.
Dropdown stängs inte när man väljer alternativ, förutom vid val av "Välja alla".

Namn Beskrivning Exempel
items : IDropdownItem[] Samling av IDropdownItem som utgör valbara alternativ [items]="[item1, item2, ...] "
disabled : boolean Defaultvärde false [disabled]="true "
showAllItemText : string Värde för att sätta vilken text det skall stå på raden för att ta bort filtrering och visa alla. Defaultvärde "Visa alla" [showAllItemText]="Visa alla alternativ"
noItemSelectedLabel : string Värde för att sätta vilken text det skall stå i dropdownen då man inte valt något värde. Defaultvärde "Välj" [noItemSelectedLabel]="Välj alternativ"
allItemsSelectedLabel : string Värde för att sätta vilken text det skall stå i dropdownen då man valt alla. Defaultvärde "Alla" [allItemsSelectedLabel]="Alla alternativ valda"
selectAllItemText : string Värde för att sätta vilken text det skall stå på alternativet för att välja alla. Defaultvärde "Välj alla" [selectAllItemText]="Välj alla alternativ"
required : boolean Anger om värde är obligatoriskt eller ej. Om required=true och inget värde valts när komponenten tappar fokus ges ett validerings fel. Vid val av värde uppdateras validering. Vid fokus indikeras att valideringsfel redigeras. För att trigga validering utifrån använd validate(). [required]="true"
validate() : IValidationResult Anropas utifrån för att validera komponenten. ValidationResult.isValid indikerar om innehåll är giltigt eller ej. ValidationResult.errorMessage ger eventuellt fel. const result = component.validate()
selectionChanged : EventEmitter<IDropdownItem[]> Event som triggas när man väljer ett alternativ i dropdownen (selectionChanged)="onMultipleSelectionChanged($event) "
readonly : boolean Om dropdownen är satt som readonly så ska en lista med valda värden visas. Om inget värde är satt så visas ingenting. [readonly]="true/false"
