Dokumentation för Monthpicker

Översikt

<vgr-monthpicker>


Ger möjlighet att välja år och månad. Ställs in med min/maxdatum som kan vara hela eller del av år.


Förändras med tema: Ja (Knapp, header samt markering av månad)

Exempel

Default formaterad

Med defaultinställningar sätts min- och maxdatum till innevarande år.

MMMM formatering

Här är selectedDate från 2015. minDate blir då default 2015, maxDate blir innevarande år.

Med max- och mindatum satt

Här är minDate satt till 2016-05-01 och maxDate är 2019-03-31. 2016 visas med samtliga månader, men januari - april är ej valbara. Hela 2017 och 2018 är valbara. 2019 visas med januari - mars valbara.

Disabled

Senast valt datum: {{lastSelectedDate | date : 'yyyy MMM'}}

API

Namn Beskrivning Exempel
selectedDateFormat defaultvärde: 'MMM yyyy', här kan man sätta hur datumet skall presenteras i dropdownen [selectedDateFormat]="'MM yyyy'"
toolTipDateFormat defaultvärde: 'MMMM yyyy', här kan man sätta hur datumet skall presenteras när man hovrar över det [toolTipDateFormat]="'MMMM'"
minDate Defaultvärdet är innevarande år. Om man förvalt ett datum innan dagens år så sätts minDate till det förvalda året. Om man satt ett förvalt datum som är efter dagens år så sätts dagens år till minDate [minDate]="minDate"
maxDate Defaultvärdet är innevarande år. Om man förvalt ett datum innan dagens år så sätts maxDate till det innevarande året. Om man satt ett förvalt datum som är efter dagens år så sätts det förvalda året till maxDate. [maxDate]="maxDate"
selectedDate Property för att kunna skicka in ett förvalt datum till komponenten [selectedDate]="forvaltDatum"
disabled true/false, defaultvärde: false. Möjlighet att sätta disabled på månadskalendern [disabled]=true
selectedDateChanged Event som triggas när man väljer ett nytt datum (selectedDateChanged)=onselectedDateChanged($event)
