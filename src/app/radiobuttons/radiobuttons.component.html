Översikt

<vgr-radio-group>


En radiogroup innehåller flera radioknappar, där bara ett alternativ är valt åt gången, val kan vara förvalda och/eller inaktiva. Vid val av alternativ triggas eventet selectedChanged. Valbara items är av typen ISelectableItem. Det är genom att sätta properties på dessa objekt som man styr beteendet.


Förändras med tema: Nej

Exempel

Du har valt alternativ: {{selectedRadioOption.displayName}}

API

Namn Beskrivning Exempel
options : ISelectableItem[] Samling av ISelectableItem som utgör valbara alternativ [options]="[item1, item2, ...]"
disabled : boolean Defaultvärde false. Sätter hela gruppen till disabled [disabled]="true"
noSelection : boolean Påverkar vad som händer om inget alternativ i options har selected: true. False innebär att första alternativet väljs. True innebär att inget alternativ väljs. [noSelection]="true"
selectedChanged : EventEmitter<ISelectableItem[]> Event som triggas när man väljer ett nytt alternativ i radiogruppen (selectedChanged)="onSelectedRadioOptionChanged($event)"
