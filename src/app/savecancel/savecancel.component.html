En panel som innehåller spara, avbryt samt låsknapp. Spara och avbryt är inaktiverade i låst läge. De kan aktiveras genom att låsa upp. När användaren sparar triggas eventet onSave, vid avbryt triggas onCancel. Om man låser upp och sedan låser är det samma som att spara.

Senaste händelse: {{saveCancelMessage}}

Konfiguration

Namn Beskrivning Exempel
enabled Tveksam property
secondary true/false, defaultvärde false. Sätter om knapparna skall ha secondary utseende eller inte. [checked]=True
onCancel Event som triggas när man trycker på avbryt (onCancel)="onCancel()"
onSave Event som triggas när man trycker på spara (onSave)="onSave()"
onUlock Event som triggas när man trycker på låset (onUlock)="onUlock()"
