Dokumentation för save/cancel

Översikt

<vgr-save-cancel>


En panel som innehåller spara, avbryt samt låsknapp. Spara och avbryt är inaktiverade i låst läge. De kan aktiveras genom att låsa upp. När spara eller avbryt sedan trycks ned, låses panelen igen. Att låsa panelen i upplåst läge innebär att spara.



Förändras med tema: Ja

Exempel


Senaste händelse:  {{saveCancelMessage}}

API

Namn Beskrivning Exempel
unlocked : boolean True = låset upplåst, spara och abryt är aktiva. False = låset är låst, spara och avbryt är inaktiva. [unlocked]="true"
secondary : boolean Om knappar ska vara sekundärknappar eller ej [secondary]="true"
cancel : EventEmitter Triggas då användaren klickar på avbryt
save : EventEmitter Triggas då användaren klickar på spara
unlock : EventEmitter Triggas då användaren låser upp låset
