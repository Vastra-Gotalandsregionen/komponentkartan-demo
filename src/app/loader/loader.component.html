Dokumentation för loader

Översikt

<vgr-loader> används för att visa att en åtgärd pågår. Loader går alltid fulla varv innan stopp. Den finns i två olika storlekar, den sätts av via small och är default false.


Förändras med tema: Nej

API

Namn Beskrivning Exempel
active : boolean Anger om loader snurrar eller ej. Minsta tid som loader visas är 1 sekund. Inaktivering kan därför fördröjas. [active]="true"
small : boolean Visar loadern i liten storlek [small]="true"
