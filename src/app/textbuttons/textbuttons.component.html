Dokumentation för knappar

Översikt

<vgr-button> finns i två olika utföranden, primär och sekundär. Det sätts via egenskapen secondary, den är default false.

Klicka här Klicka här

Förändras med tema: Ja

Exempel

Primärknappar

OK {{buttonDisabled ? 'Aktivera' : 'Inaktivera'}} {{buttonDisabled ? 'Inaktiv' : 'Aktiv'}}

Sekundärknappar

OK {{buttonSecondaryDisabled ? 'Aktivera' : 'Inaktivera'}} {{buttonSecondaryDisabled ? 'Inaktiv' : 'Aktiv'}}


Senaste knapptryck: {{lastButtonPressed}}

API

Namn Beskrivning Exempel
secondary : boolean Visa sekundärt utseende. Default är false. [secondary]="true"
disabled : boolean Inaktivera knapp [disabled]="true"
click Event när knapp trycks ner (click)="OnClick()"
