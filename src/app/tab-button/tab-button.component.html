<vgr-tab-button> är avsedd att användas för att sortera och växla mellan olika innehåll.
I kombination med <vgr-tab-button-group> kan tabbarna navigeras med piltangenterna.



Förändras med tema: ja



Namn Beskrivning Exempel
<vgr-tab-button>
disabled: boolean Inaktivera tab button. Förvalt värde är false. [disabled]="true"
ariaLabel: string Värde för aria-label. Förvalt värde är "[text] tab button", där [text] är text content för <vgr-tab-button>. [ariaLabel]="En alternativ beskrivning."
active: boolean Värde för active. Förvalt värde är false. [active]="true"
tabId: string Värde för tabId. Förvalt värde är content för <vgr-tab-button>. [tabId]="true"
 
focus() Metod för att flytta fokus till komponenten this.myTabButton.focus()
 
<vgr-tab-button-group>
width: string Värde för den totala bredd för alla segment i gruppen. Förvalt värde är "" vilket innebär fullbredd. [width]="'200px'"
 

<vgr-tab-button> har ett intern element av typen button och har alla dess WCAG egenskaper. Default sätts attr.aria-label till content för <vgr-tab-button>. Om input variabeln [disabled] sätts till true sätts även aria.aria-disabled till true
