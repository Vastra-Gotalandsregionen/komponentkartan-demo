Dokumentation för Title-value

Översikt

<vgr-title-value> används för att skapa title-value layouter, t.ex formulär.
Placera title-value komponenterna i <vgr-title-value-layout> .

Förändras med tema: nej

 API - title-value
Namn Beskrivning Exempel
slim : boolean Om title-value är satt som slim så kommer title-value-raden ta så lite plats som möjligt i höjdled.
Detta ska användas för en title-value komponent som endast visar readonly fält. 		[readonly]="true/false"
API - title-value-heading
Namn Beskrivning Exempel
width : number Bredd på title-value-heading anges som proportioner i heltal, där siffran motsvarar kolumnens storlek i proportion till summan av alla kolumners storlekt. Varje kolumn måste ha en width satt och summan av width ska vara 100% av brädden. [width]="1"
API - title-value-container
Namn Beskrivning Exempel
width : number Bredd på title-value-container anges som proportioner i heltal, där siffran motsvarar kolumnens storlek i proportion till summan av alla kolumners storlekt. Varje kolumn måste ha en width satt och summan av width ska vara 100% av brädden. [width]="3"
