Dokumentation för datepicker

Översikt

<vgr-datepicker> är en kalender där användaren kan välja ett datum. I kalendern visas valt datum (satt av användaren eller av systemet) och dagens datum.



Förändras med tema: ja

Exempel

Default formatering

Valt datum default:

{{datepicker_selectedDate1 | date : 'yyyy MM dd'}}

Format yyyy MMM dd

Valt datum:

{{datepicker_selectedDate2 | date : 'dd MMMM yyyy'}}

Format yyyy på tooltip

Valt datum:

{{datepicker_selectedDate3 | date : 'dd MMM yyyy'}}

Avstängd datepicker

API

Namn Beskrivning Exempel
minDate: Date Sätter vilket datum kalendern börjar på. Sätts inget datum är standard första dagen i januari månad för innevarande år. [minDate]=someDate
maxDate: Date Sätter vilket datum kalendern slutar på. Sätts inget datum är standard sista dagen i december månad för innevarande år. [maxDate]=someDate
selectedDate: Date Valt datum som visas i kalendern. När kalendern instansieras och man öppnar den första gången så positioneras man på den månaden selectedDate är valt. Om inget datum är valt så positioneras man på dagens datum. [selectedDate] = someDate
disabled: boolean Inaktiverar kalendern. [disabled]=true/false
selectedDateFormat: string Datumformatet på valt datum i dropdownen. [selectedDateFormat]=dateFormat
tooltipDateFormat: string Datumformatet på valt datum i dropdownens tooltip. [selectedDateFormat]=dateFormat
selectedDateChanged = EventEmitter<date> Event som trigas när man väljer ett datum i kalendern. (selectedDateChanged)="someDateProperty=$event"
