Dokumentation för Datepicker

Översikt

<vgr-datepicker> är en kalender där användaren kan välja ett datum. I kalendern visas valt datum (satt av användaren eller av systemet) och dagens datum.



Förändras med tema: ja

Exempel

Default formatering

Valt datum default:

{{datepicker_selectedDate1 | date : 'yyyy MM dd'}}

Format yyyy MMM dd

Valt datum:

{{datepicker_selectedDate2 | date : 'dd MMMM yyyy'}}

Format yyyy på tooltip

Valt datum:

{{datepicker_selectedDate3 | date : 'dd MMM yyyy'}}

Avstängd datepicker

API

Namn Beskrivning Exempel
minDate: Date Beskrivning [propp]=" 'value' ", default
maxDate: Date Beskrivning [propp]=" 'value' "
selectedDate?: Date Beskrivning [propp]=" 'value' "
disabled: boolean Beskrivning [propp]=" 'value' "
selectedDateFormat: string Beskrivning, default yyyy-MM-dd [propp]=" 'value' "
tooltipDateFormat: string Beskrivning, default yyyy-MM-dd [propp]=" 'value' "
selectedDateChanged = new EventEmitter Beskrivning [propp]=" 'value' "

Tillgänglighet

<vgr-checkbox> har en intern div vars roll är satt till checkbox. Förutom rollen så används attributen: attr.aria-disabled (inaktiv/aktiv), attr.aria-checked (kryssad/ej ikryssad) och attr.aria-labelledby (pekning till interna label i <vgr-checkbox>)
