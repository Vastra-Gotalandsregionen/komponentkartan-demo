Notifiering som ex. kan placeras i grid.
<vgr-notification>

Man kan välja att en ikon automatiskt läggs till enligt standardmall för hur notifieringar ser ut. Om man vill ha en speciell ikon som inte finns bland standardnotifieringarna kan man lägga in en egen ikon.

Förändras med tema: nej

Namn Beskrivning Exempel
autoAddTypeIcons : boolean Anger om man vill att en ikon automatiskt läggs till. Defaultvärde är false. [autoAddTypeIcons]="false"
type : string Förutsatt att autoAddTypeIcons är true kan man välja typ av notifiering. Tillåtna värden är 'success', 'error', 'comment' och 'default'. Default är 'default'. [type]="'error'"
Ej aktuellt.
