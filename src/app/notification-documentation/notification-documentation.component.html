<vgr-notification>
Notifiering som ex. kan placeras i grid eller helt fristående.

Förändras med tema: nej

Standalone notifieringar


En alert av typen success <vgr-notification [type]="'success'" [standalone]="true">
  En alert av typen success
</vgr-notification>

En alert av typen error med bredd satt <vgr-notification [type]="'error'" [standalone]="true" [width]="'300px'">
  En alert av typen error med bredd satt
</vgr-notification>

En alert av typen warning med bredd och höjd satt <vgr-notification [type]="'warning'" [standalone]="true" [width]="'500px'" [height]="'150px'">
  En alert av typen warning med bredd och höjd satt
</vgr-notification>

 <vgr-notification [type]="'info'" [standalone]="true">

En alert av typen error utan ikon <vgr-notification [type]="'error'" [standalone]="true" [noIcon]="true">

</vgr-notification>

En alert av typen error med custom ikon <vgr-notification [type]="'error'" [standalone]="true" [noIcon]="true">
<vgr-icon [solid]="true" color="error" icon="star"></vgr-icon> En alert av typen error med custom ikon
</vgr-notification>

Namn Beskrivning Exempel
standalone : boolean Skall notifieringen använndas utanför vgr-grid [standalone]="'true'"
type : string Styr vilken notifiering man vill ha (ikon och färg som automatiskt läggs till). Tillåtna värden är 'success', 'error', 'warning', 'info' och 'comment'. [type]="'error'"
width : string Sätter bredden på notifieringen [width]="'300px'"
height : string Sätter höjden på notifieringen. Det som sätts är min-height då den behöver kunna växa åt något håll ifall innehåll är för stort [height]="'100px'"
noICon : boolean Om man ej vill ha ikonen som annars följer med automatiskt när man anger type [noICon]="'true'"
