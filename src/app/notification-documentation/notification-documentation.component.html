<vgr-notification>
Notifiering som ex. kan placeras i grid.

Man kan välja att en ikon automatiskt läggs till enligt standardmall för hur notifieringar ser ut. Om man vill ha en egenkomponerad notifiering kan man göra detta gemom att deklarativt lägga till en vgr-icon samt sätta ett giltig värde på [borderColor].

Förändras med tema: nej

Standardnotifiering
<vgr-notification [type]="'error'" >Ett felmeddelande. </vgr-notification>

Egenkomponerad
<vgr-notification [borderColor]="'secondary'" >
  <vgr-icon [icon]="'star'"></vgr-icon> Ett annat meddelande
</vgr-notification>
Namn Beskrivning Exempel
type : string Styr vilken standardnotifiering man vill ha (ikon och färg som automatiskt läggs till). Tillåtna värden är 'success', 'error' och 'comment'. standardnotifiering läggs ändast till om [type] har ett giltigt värde. [type]="'error'"
borderColor : string Om man vill lägga till en egenkomponerad notifiering måste man välja färg bordern manuellt. Tillåtna värden är 'text', 'success', 'error', 'error-secondary', 'primary', 'secondary', 'dimmed', 'light', 'disabled' . [borderColor]="'disabled'"
