API

CSS-Klass Beskrivning Exempel
.information-panel Sätts på element som skall vara en panel Kombinera med klasserna .full, .one-third, .two-thirds eller half för att sätta storlek. Notera att panelerna då måste ligga i en .row-layout.
.information-panel__content Innehåll i panelen, innehåller header, information samt footer
.information-panel__header Headerelement, innehåller rubrik och ev tidsstämpel
.information-panel__header__title Panelens rubrik, visas till vänster i rubriken
.information-panel__header__timestamp Tidsstämpel, visas till höger i header
.information-panel__information Plats för text eller annat innehåll
.information-panel__footer Visas längst ned i panelen
