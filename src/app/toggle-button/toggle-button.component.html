<vgr-toggle-button> är avsedd att användas för att växla mellan olika states.
I kombination med <vgr-toggle-button-group> kan knappsegmenten navigeras med piltangenterna.

Namn Beskrivning Exempel
<vgr-toggle-button>
disabled: boolean Inaktivera toggle button. Förvalt värde är false. [disabled]="true"
ariaLabel: string Värde för aria-label. Förvalt värde är "[text] toggle button", där [text] är text content för <vgr-toggle-button>. [ariaLabel]="En alternativ beskrivning."
pressed: boolean Värde för pressed (knappens state). Förvalt värde är false. [pressed]="false"
<vgr-toggle-button-group>
width: string Värde för den totala bredd för alla segment i gruppen. Förvalt värde är "" vilket innebär fullbredd. [width]="'200px'"

<vgr-toggle-button> har ett intern element av typen button och har alla dess WCAG egenskaper. Internt sätts attr.aria-pressed till värdet av input variabeln [pressed]. Default sätts attr.aria-label till "[text] toggle button", där [text] är content för <vgr-toggle-button>. Om input variabeln [disabled] sätts till true sätts även aria.aria-disabled till true
