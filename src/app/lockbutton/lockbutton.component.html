Låsknappen är en "toggle-knapp", som kan vara låst eller upplåst.


Senaste händelse: {{lockMessage}}

Konfiguration

Namn Beskrivning Exempel
unlocked Om knappen ska vara upplåst. True/false, default är false. [unlocked]="true"
onUnlocked Event som triggas när knappen låses upp
onLocked Event som triggas när knappen låses
