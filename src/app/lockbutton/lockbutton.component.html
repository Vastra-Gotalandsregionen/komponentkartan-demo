Dokumentation för lock button

Översikt

<vgr-lock-button>


Låsknappen är en "toggle-knapp", som kan vara låst eller upplåst.

Förändras med tema: Ja (bakgrundsfärg)

Exempel



Senaste händelse:  {{lockMessage}}

API

Namn Beskrivning Exempel
unlocked : boolean Om knappen ska vara upplåst. True/false, default är false. [unlocked]="true"
disabled : boolean Om knappen ska vara inaktiverad. [disabled]="true"
onLockChanged : EventEmitter<boolean> Event som triggas när knappen låses eller låses upp. True innebär låst, false upplåst.
