Översikt

<vgr-list> är en lista bestående av vgr-list-item. En lista med header och expanderbara rader, som används till ex. listor av kort. Rubrikraden är fast positionerad och kan inte scrollas bort.

Förändras med tema: Nej

Exempel

Se exempel (på egen sida)

API - vgr-list

Namn Beskrivning Exempel
allowMultipleExpandedChildren : boolean Om flera rader skall kunna vara expanderade samtidigt, default är false. [allowMultipleExpandedChildren]="true"
flexibleHeader : boolean Kan man sätta om headern skall följa med i scrollningen, default är false. [flexibleHeader]="true"

API - vgr-list-item

Namn Beskrivning Exempel
expanded : boolean Anger om raden skall vara expanderad eller inte. [expanded]="false"
notification : RowNotification Ger möjligheten att lägga en notifiering på raden. En NotificationType har en icon (Error, Warning, Information) och en typ. Notifieringar kan visas hela tiden - NotificationType.Permanent, visas i samband med att raden minimeras - NotificationType.ShowOnCollapse eller när raden tas bort - NotificationType.ShowOnRemove [notification]="{{'{'}} message: 'Personen är inaktiv', icon: NotificationIcon.Exclamation, type: notificationTypes.Permanent{{'}'}}" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: notificationTypes = NotificationType
expandedChanged : EventEmitter<boolean> Event som triggas när raden expanderar (expandedChanged)="onExpandedChanged($event)"
notificationChanged : EventEmitter<RowNotification> Event som triggas när notifiering ändras (notificationChanged)="onNotificationChanged($event)"
