Översikt

<vgr-list> är en lista bestående av <vgr-list-item>. En lista med header och expanderbara rader, som används till ex. listor av kort. Rubrikraden är fast positionerad och kan inte scrollas bort.


Förändras med tema: Nej

Enkel lista
Lista med notifieringar

API - vgr-list

Namn Beskrivning Exempel
allowMultipleExpandedChildren : boolean Om flera rader skall kunna vara expanderade samtidigt, default är false. [allowMultipleExpandedChildren]="true"
flexibleHeader : boolean Kan man sätta om headern skall följa med i scrollningen, default är false. [flexibleHeader]="true"

API - vgr-list-item

Namn Beskrivning Exempel
expanded : boolean Anger om raden skall vara expanderad eller inte. [expanded]="false"
notification : RowNotification Ger möjligheten att lägga en notifiering på raden. En NotificationType har en icon (Error, Warning, Information) och en typ. Notifieringar kan visas hela tiden - NotificationType.Permanent, visas i samband med att raden minimeras - NotificationType.ShowOnCollapse eller när raden tas bort - NotificationType.ShowOnRemove [notification]="{{'{'}} message: 'Personen är inaktiv', icon: NotificationIcon.Exclamation, type: notificationTypes.Permanent{{'}'}}" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: notificationTypes = NotificationType
expandedChanged : EventEmitter<boolean> Event som triggas när raden expanderar (expandedChanged)="onExpandedChanged($event)"
notificationChanged : EventEmitter<RowNotification> Event som triggas när notifiering ändras (notificationChanged)="onNotificationChanged($event)"

API - vgr-list-column-header

Namn Beskrivning Exempel
text : string Kolumnens rubrik [text]="'Förnamn'"
sortDirection : SortDirection Anger åt vilket håll kolumnen sorteras vid initiering SortDirection.None, default - ingen sortering SortDirection.Ascending sortera stigande SortDirection.Descending sortera fallande [sortDirection]="sortDirection.Ascending" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: sortDirection = SortDirection
sortKey : string Nyckel som används för att identifiera kolumnen vid sorteringsevent. [sortKey]="'firstName'"
width : number vgr-list är baserad på en 20 kolumns layout. Varje kolumn måste ha en width satt och summan av width ska vara 20 för vgr-list.
Bredden på rubrik-kolumnerna ärvs ner till sina värde-kolumner		 [width]="20"
align : string Man kan sätta en alignment på kolumnerna. Defaultvärde är left. Giltiga värden är right, left och center.
Om man sätter ett felaktigt värde så sätts defaultvärdet.
Justeringen på rubrik-kolumnerna ärvs ner till sina värde-kolumner		 [align]="'right'"
sortChanged : EventEmitter<SortDirection> Event som triggas när sortering sker i kolumnen (sortChanged)="onSortChanged($event)"

API - vgr-list-header

Innehåller ett antal vgr-list-column-header element och utför vissa åtgärder på dem.
Namn Beskrivning Exempel
sortChanged : EventEmitter<SortChangedArgs> Event som triggas när sortering sker i någon av headerns kolumner (sortChanged)="onSortChanged($event)"

API - vgr-list-column

Namn Beskrivning Exempel
text : string Kolumnens textinnehåll [text]="Sverige"

API - vgr-list-column-trashcan

En variant av vgr-list-column som innehåller en papperskorg.
Namn Beskrivning Exempel
disabled : boolean Inaktiverar papperskorgen, den blir utgråad. [disabled]="true"
delete : EventEmitter Event som triggas då användaren trycker på papperskorgen. (delete)="onDelete()"

API - vgr-list-column-checkbox

En variant av vgr-list-column som innehåller en vgr-checkbox.
Namn Beskrivning Exempel
disabled : boolean Inaktiverar checkboxen, den blir utgråad. [disabled]="true"
checked : boolean Inaktiverar checkboxen, den blir utgråad. [disabled]="true"
checkedChanged : EventEmitter<boolean> Event som triggas då användaren trycker på checkboxen. (checkedChanged)="onCheckChanged($event)"
