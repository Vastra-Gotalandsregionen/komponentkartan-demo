Under "kodexempel" finns exempel på hur dropdown-komponenten kan användas i Angular Reactive Forms:
Reactive Forms exempel
Namn Beskrivning Exempel
items : IDropdownItem[] Samling av IDropdownItem som utgör valbara alternativ [items]="[item1, item2, ...]"
disabled : boolean Defaultvärde false [disabled]="true"
showAllItemText : string Värde för att sätta vilken text det skall stå på raden för att ta bort filtrering och visa alla. Defaultvärde"Visa alla" [showAllItemText]="Visa alla alternativ"
noItemSelectedLabel : string Värde för att sätta vilken text det skall stå i dropdownen då man inte valt något värde [noItemSelectedLabel]="Välj alternativ"
required : boolean Anger om värde är obligatoriskt eller ej. Om required=true och inget värde valts när komponenten tappar fokus ges ett validerings fel. Vid val av värde uppdateras validering. Vid fokus indikeras att valideringsfel redigeras. För att trigga validering utifrån använd validate(). [required]="true"
validate() : IValidationResult Anropas utifrån för att validera komponenten. ValidationResult.isValid indikerar om innehåll är giltigt eller ej. ValidationResult.errorMessage ger eventuellt fel. const result = component.validate()
selectedItemChanged : EventEmitter<IDropdownItem[]> Event som triggas när man väljer ett nytt alternativ i dropdownen (selectedItemChanged)="onselectedItemChanged($event)"
readonly : boolean Om dropdownen är satt som readonly så ska bara textfält med valt värde visas. Om inget värde är satt så visas ingenting. [readonly]="true/false"
