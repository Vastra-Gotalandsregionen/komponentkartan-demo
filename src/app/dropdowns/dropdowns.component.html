Dokumentation för dropdown

Översikt

<vgr-dropdown> innehåller en lista över valbara alternativ.
- Kompletteras med filter och ett "visa alla" alternativ över 20 items.
- Kompletteras med scroll vid över 8 items. Vid val av alternativ triggas eventet selectedItemChanged. Valbara items är av typen IDropdownItem.

Förändras med tema: Ja

Exempel

8 items

9 items

25 items

200 items


Senaste val: {{lastSingleSelection}}

API

Namn Beskrivning Exempel
items : IDropdownItem[] Samling av IDropdownItem som utgör valbara alternativ [items]="[item1, item2, ...] "
disabled : boolean Defaultvärde false [disabled]="true "
showAllItemText : string Värde för att sätta vilken text det skall stå på raden för att ta bort filtrering och visa alla. Defaultvärde "Visa alla" [showAllItemText]="Visa alla alternativ"
noItemSelectedLabel : string Värde för att sätta vilken text det skall stå i dropwnen då man inte valt något värde [noItemSelectedLabel]="Välj alternativ"
selectedItemChanged : EventEmitter<IDropdownItem[]> Event som triggas när man väljer ett nytt alternativ i dropdownen (selectedItemChanged)="onselectedItemChanged($event) "
