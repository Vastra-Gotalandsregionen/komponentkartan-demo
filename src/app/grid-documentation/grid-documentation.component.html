Dokumentation för Grid

Översikt

En tabell med header och expanderbara rader med är möjlighet till sortering och paginering.

Man kan lägga till notifieringar på radnivå samt i toolaren ovanför headern. Detta görs genom att deklarativt lägga till komponentenvgr-notifiaction.

<vgr-grid>


Förändras med tema: nej

API

<vgr-grid>

Namn Beskrivning Exempel
allowMultipleExpandedRows : boolean Om flera rader skall kunna vara expanderade samtidigt, default är false. [allowMultipleExpandedRows]="true"
stickyHeader : boolean Kan man sätta om headern skall vara sticky, default är false. [stickyHeader]="true"
zebraLayout : boolean Kan man sätta om man vill att griden ska ha olika färg på varannan rad (Zebra-randigt). Default är false. [zebraLayout]="true"
toggleAnimation: string Anger hastigheten på anmimationen för expanderad och kollapsa innehållet. Tillåtna värden 'none' | 'slow' | 'medium' | 'fast', default är 'medium' [toggleAnimation]="'slow'"
pages : number Om paginering ska visas anges antal sidor som ska hanteras. Giltigt värde är ett eller större. Paginering visas endast om värdet är strörre än ett. [pages]="10"
activePage : number Styr aktiv sida om paginering är vald ([pages] är satt). Giltigt värde är ett till [pages]. [activePage]="1"
showLoader : boolean Visar en loader under headern när den är satt till true. [showLoader]="true". Default värde är false.
ariaLabel : string Beskrivande text som läses upp av skärmläsare. [ariaLabel]="'Enhetslista'"
pageChanged :
EventEmitter <number>		 Event som triggas när aktiv sida ändras. (pageChanged)="onPageChanged($event)"

<vgr-grid-header-toolbar>

Kan innehålla valfritt innehåll. Innehållet blir sticky om stickyHeader är true.

<vgr-grid-header>

Innehåller ett antal vgr-grid-header-column element.

Namn Beskrivning Exempel
sortChanged : EventEmitter<SortChangedArgs> Event som triggas när sortering sker i någon av headerns kolumner (sortChanged)="onSortChanged($event)"

<vgr-gridheader-column>

Kan innehålla valfritt innehåll.

Namn Beskrivning Exempel
width : number Kolumners bredd anges som proportioner i heltal, där siffran motsvarar kolumnens storlek i proportion till summan av alla kolumners storlekt. Varje kolumn måste ha en width satt och summan av width ska vara 100% av brädden. [width]="200"
align : string Man kan sätta en alignment på kolumnerna. Defaultvärde är left. Giltiga värden är right, left och center.
Om man sätter ett felaktigt värde så sätts defaultvärdet 		[align]="'right'"
sortKey : string Nyckel som används för att identifiera kolumnen vid sorteringsevent. [sortKey]="'firstName'"
sortDirection : GridSortDirection Anger åt vilket håll kolumnen sorteras vid initiering GridSortDirection.None, default - ingen sortering GridSortDirection.Ascending sortera stigande GridSortDirection.Descending sortera fallande [sortDirection]="sortDirection.Ascending" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: sortDirection = GridSortDirection
sortChanged : EventEmitter<SortDirection> Event som triggas när sortering sker i kolumnen (sortChanged)="onSortChanged($event)"
 
focus() Metod för att flytta fokus till komponenten this.myGridHeaderColumn.focus()

<vgr-grid-row>

Namn Beskrivning Exempel
expanded : boolean Anger om raden skall vara expanderad eller inte. [expanded]="false"
preventCollapse : boolean Anger om raden tillåts att kollapsa. [preventCollapse]="true"
expandedChanged : EventEmitter<boolean> Event som triggas när raden expanderar (expandedChanged)="onExpandedChanged($event)"
expandPrevented : EventEmitter<boolean> Event som triggas när raden försöker expandera men hindras (expandPrevented)="onExpandPrevented($event)"
collapsePrevented : EventEmitter<boolean> Event som triggas när raden försöker kollapsa men hindras (collapsePrevented)="onCollapsePrevented($event)"
 
focus() Metod för att flytta fokus till komponenten this.myGridRow.focus()

Tillgänglighet

Navigering i griden:

- Enter och Space: När fokus är på listradens header togglas listraden expanderad / kollapsad.
- Nedåtpil: Om fokus ligger på en listrad eller en headerkolumn flyttar fokus till raden nedanför. Om fokus ligger på den sista raden flyttas fokus till den första raden.
- Uppåtpil: Om fokus ligger på en listrad, flyttar fokus till raden ovanför. Om fokus ligger på första raden, flyttas fokus till den sista raden.
- Home: När fokus ligger på en listrad, flyttar fokus till första raden.
- End: När fokus ligger på en rad flyttar fokus till sista raden.
