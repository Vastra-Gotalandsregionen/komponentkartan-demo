<vgr-pagination> beskrivning av komponenten....
Max sju sidnummer visas, sedan punktas sidor i mitten.
Första och sista sidnumret visas alltid.
Piltangenterna används för att förflytta sig inom pagineringen.
Enter/space byter sida.


Förändras med tema: ja

Kodexempel:


Namn Beskrivning Exempel
pages : number Om paginering ska visas anges antal sidor som ska hanteras. Giltigt värde är ett eller större. Paginering visas endast om värdet är strörre än ett. [pages]="10"
activePage : number Styr aktiv sida om paginering är vald ([pages] är satt). Giltigt värde är ett till [pages]. [activePage]="1"
pageChanged :
EventEmitter <number>		 Event som triggas när aktiv sida ändras. (pageChanged)="onPageChanged($event)"

<vgr-pagination> består internt av ett antal element av typen button och har alla dess WCAG egenskaper. Internt sätts attr.aria-label för de olika knapparna exempelvis till "Du är på sidan x av y" eller om man navigerar med piltangenterna "Gå till sida x av y".
