<vgr-radiobutton-group>


En radiogroup innehåller flera radioknappar, där bara ett alternativ är valt åt gången, val kan vara förvalda och/eller inaktiva. Vid val av alternativ triggas eventet selectedChanged. Man kan placera valfria objekt i sina items, eller bara sätta en text. Om man lägger objekt i sina items så är det bra att även sätta ett värde för sitt item så gruppen vet om detta.

Under "kodexempel" finns fler exempel på hur radiogroup-komponenten kan användas i Angular Reactive Forms:
Reactive Forms exempel

vgr-radiobutton-group

Namn Beskrivning Exempel
formControlName : sträng Möjlighet att koppla ett form till gruppen [formControlName]="'cars'"
disabled : boolean Sätter hela gruppen till disabled.Defaultvärde false [disabled]="true"
showValidation Talar om huruvida komponenten skall visa felmeddelanden. Default är true (visar felmeddelanden) [showValidation]="false"
required : boolean Anger om radiogruppen är obligatorisk eller inte. Defaultvärde false (ej obligatorisk). [required]="true"
vertical : boolean Anger om radiogruppen ska ha horisontell eller vertikal layout. Defaultvärde false (horisontellt). [vertical]="true"
selectedChanged : EventEmitter<RadiobuttonItemComponent> Event som triggas när man väljer ett nytt alternativ i radiogruppen (selectedChanged)="onSelectedRadioOptionChanged($event)"

vgr-radiobutton-item

Namn Beskrivning Exempel
selected : boolean Om item är valt [selected]="true"
disabled : boolean Om item är disabled. Defaultvärde false [disabled]="true"
value: string | boolean Talar om vilket värde valet har. Default är textcontent på innehållet man har [value]="1"

<vgr-radiobutton-group> har en intern div vars roll är satt till radiogroup och sedan innehåller den valfritt antal <vgr-radiobutton-item> som har sin roll satt till radio. Förutom rollen så används attributen: attr.aria-checked (kryssad/ej ikryssad), attr.aria-disabled (om den är tillgänglig, både på grupp och item nivå) och attr.aria-labelledby (pekning till interna label i <vgr-radiobutton-group>)


Vid tab in till radiogruppen så gäller nedan regler:
- Om ingen av radioknapparna är valda så ställs in fokus på den första radioknappen i gruppen och den väljs
- Om en radioknapp är vald ställs fokus på den valda knappen
